# Controle-de-Estacionamento
Projeto acadêmico desenvolvido em parceria com colegas de curso. O programa simula o funcionamento de um sistema de cobrança em estacionamentos, calculando o valor a ser pago com base no tempo de permanência do veículo. A lógica considera uma cobrança proporcional ao tempo utilizado, oferecendo uma solução simples e funcional para essa aplicação.
